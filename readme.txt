== Installation ==

1. Upload 'MonoVM-Blog' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Or you can install the plugin in: "Plugins ==> Add ==> enter the name of plugin (MonoVM Blog) in search box and press Enter
4. Install plugin, activate it and enjoy of plugin
